# EcoShop — Zero-Waste Kitchen Landing

Тестовий лендінг-проєкт для курсу AI in Marketing.

**Жива версія:** _додасться після деплою на Vercel_

## Що це

Продажний лендінг для гіпотетичної української eco-subscription компанії **EcoShop**, побудований на основі контент-стратегії та блогу про zero-waste кухню. Лендінг рекламує підписку **EcoBox** — щомісячну коробку перевірених еко-товарів від UA-виробників.

## Структура

```
ecoshop-landing/
├── index.html       # Повний лендінг (HTML + inline CSS, без JS-залежностей)
└── README.md        # Цей файл
```

Один файл, нуль залежностей. Шрифти — Google Fonts (Fraunces + Inter, з підтримкою кирилиці). Зображення — Unsplash CDN.

## Стек

- **HTML + CSS** — поточна версія (статичний лендінг)
- **Vercel** — хостинг і деплой
- **GitHub** — версіонування

**Плани:** міграція на Next.js + Supabase (waitlist форма + база підписників) на наступному етапі.

## Контент

Продакт-секція побудована на основі статті з контент-плану:
- 7 zero-waste замін на кухні
- Розрахунки економії (до ₴16,000/рік для сім'ї з 3 осіб)
- 2 тарифи EcoBox (₴499 / ₴799 на місяць)

## Локальний запуск

Просто відкрий `index.html` у браузері. Жодних build-команд не потрібно.

## Контекст

Проєкт розроблений як частина навчального курсу. Усі цифри та виробники — частково реальні (Натуралаб, Мильна Мануфактура), частково ілюстративні. Це не діючий e-commerce.
